# Instagram Clone

A responsive Instagram-inspired frontend built using pure HTML and CSS.

This project recreates the core feel of Instagram’s feed experience while focusing heavily on frontend structure, layout design, responsive behavior, and interactive UI elements without using frameworks or JavaScript.

The goal of this project was to strengthen frontend development skills through building a realistic social media interface from scratch.

---

## Features

### Responsive Navigation Sidebar

- Fixed sidebar navigation
- Hover underline animations
- Font Awesome icons
- Instagram-inspired layout structure

### Stories Section

- Horizontally scrollable stories
- Circular story previews
- Fullscreen story view using CSS `:target`
- Overlay story experience without JavaScript

### Feed System

- Multiple post layouts
- Single-image posts
- Multi-image carousel posts
- Video posts with autoplay and looping support

### Interactive UI Elements

- Like button toggle animation
- Hover-based comments popup
- Bookmark and share icons
- Smooth scrolling behavior

### Custom Styling

- Dark-mode Instagram-inspired theme
- Custom hover effects
- Flexbox layouts
- Scroll snapping
- Clean spacing and responsive alignment

## UI Highlights

### Story Modal System

The fullscreen story system was built entirely using HTML anchors and CSS `:target` selectors instead of JavaScript.

### Multi-Image Posts

Horizontal image scrolling was implemented using:

- `overflow-x`
- `scroll-snap-type`
- hidden scrollbars
- flexbox layouts

### Interactive Like Button

The heart toggle effect uses:

- hidden checkboxes
- sibling selectors
- CSS transitions
